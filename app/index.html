Nav Generator

How it works

  1. Select the category level that you are working with in the drop down menu. (e.g L-1, L-2, L-3)
  2. Click the Generate button after you enter the number of categories you want after you have selected the category level.
  3. Fill out all of the information for each respective category.
    • The title would be the name of the category (e.g If we have an L-2 named Denim Everything then the title would be that).
    • The category url would be the name of the url section (e.g $httpUrl('Search-Show','cgid','denim-everything')$)
  4. Click the Parse HTML button once you have entered all of the data.
  5. The HTML will now be available for you on the right to copy and paste into Demand Ware Business Manager.
  6. Hit the Reset button to clear all the data if you want to start over.
